# Adora

Landing page da Adora — a ferramenta de copy por IA sem censura pra tráfego pago. VSLs, criativos e lateralização de hooks, do white ao black hat, em qualquer idioma e qualquer país.

## Stack

Site estático (HTML + CSS + JS puro, sem build step).

## Rodar localmente

```bash
npx serve .
```

ou

```bash
python3 -m http.server 4321
```
